# GOD’S NOTE～神の恵み～

自分専用のライフログ / ジャーナリングPWAです。

## ローカルで開く

```powershell
node server.mjs
```

PCでは次を開きます。

```text
http://127.0.0.1:4173
```

## スマホで試す

同じWi-Fi内のスマホから一時的に見る場合は、PC側で次のように起動します。

```powershell
$env:HOST="0.0.0.0"; node server.mjs
```

そのあと、PCのローカルIPアドレスを使ってスマホで開きます。

```text
http://PCのIPアドレス:4173
```

スマホのホーム画面に入れるPWAとして安定運用するなら、GitHub PagesやNetlifyなどのHTTPS環境に置くのがおすすめです。

## 入っている機能

- 月間カレンダー
- 日別の朝チェックイン
- 朝チェックイン時間の自動記録
- 気分、眠気、やる気の記録
- バッドエンド / ハッピーエンドの朝ジャーナリング
- 睡眠、本業、副業、運動の記録
- 複数種目の運動記録
- METsによる運動消費カロリーの簡易推定
- 夜チェックアウト
- 時間に関係なく書ける雑記ジャーナル
- 週次 / 月次の朝の気分・眠気・やる気グラフと集計
- JSONエクスポート

データはブラウザのlocalStorageに保存されます。
Firebase同期を設定すると、同じGoogleアカウントで復元できるクラウド保存も使えます。設定手順は `FIREBASE_SETUP.md` を見てください。

## 変更が反映されないとき

PWAのキャッシュが残っている場合があります。ブラウザで再読み込みしてください。

```text
Ctrl + F5
```
